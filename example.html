miniForm.jquery.js
A jQuery minimal form plugin

Demo

The Code

				$('#demo-form').miniForm();

Inspired by Minimal Form Interface on codrops. This plugin allows you to specify your form, input, and label elements as you see fit then when called the plugin wraps the existing form with classes and list tags and adds the control elements and position styles(unless otherwise specified).




Allows for passed in parameters for customization.



			$('.demo-form').miniForm({
				css:"css", // Default css: "jquery"
				note: {
					required: 'This is the alert text when a user triggers the required feild alert',
					empty: 'Alert for calling the plugin on an empty form',
				},
			});
More to come.
