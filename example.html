miniForm.js
A minimalist take on forms.

Demo

Working on my user system I spent a fair amount of time on the login and registration forms. In the process I created this form interface for new user registration. I was looking to create a reusable form interface for similar projects in the future. Currently, this plugin only works for text type inputs but I plan to implement more in the future.

Inspired by Minimal Form Interface on codrops. The form is generated from a JavaScript object passed into the plugin when it is called on a web page element. You can specify a string of attributes for each form field.
